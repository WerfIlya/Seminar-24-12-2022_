## Ветки в Git

Для создания ветки в введите команду:

```
git branch НазваниеВетки
```

Посмотреть список веток:

```
git branch
```

или:

```
git log --graph --all
```

Перейти на ветку:

```
git checkout НазваниеВетки
```
